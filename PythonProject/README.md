# AI Bin Sorter 🗑️

## Quick Start (2 minutes)

1. **Install**: `pip install -r requirements.txt`
2. **Model**: Place `best.pt` in `models/`
3. **Config**: Update `ESP32_IP` in `app.py`
4. **Run**: `python run.py`
5. **Open**: `http://YOUR_COMPUTER_IP:5000` on phone browser

## ESP32 Setup
- Flash your ESP32-CAM with the Arduino code (Ideatestardruino.ino)
- Note the ESP32 IP address (Serial Monitor)
- Update `ESP32_IP` in app.py
